## 📊 Project Components

### 🔍 Salary Analysis

- **Objective:** Understand salary distribution and identify key trends across job roles.
- Performed exploratory analysis using Excel functions, charts, and pivot tables.
- Focused on patterns related to job levels, industries, and compensation metrics.
- Generates insights that can help job seekers and analysts compare salaries across segments.

### 📈 Salary Dashboard

- An **interactive Excel dashboard** that visualizes key findings from the analysis.
- Includes:
  - Salary distribution by job category.
  - Summary metrics highlighting top-earning groups.
  - Visual charts for easier interpretation.
- Accessible in both Excel (`.xlsx`) and as a preview image (`.png`) on this repository.
